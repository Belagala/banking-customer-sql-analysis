# Banking Customer Data Analysis using SQL

## Project Overview
This project analyzes banking customer transaction data using SQL. The objective is to understand regional transaction trends, account performance, and overall customer activity.

## Tools Used
SQL  
Excel (for dataset creation)

## Dataset Description
The dataset contains banking transaction records with the following fields:

- Customer ID
- Region
- Account Type
- Transaction Amount
- Transaction Date

## Business Questions

1. Which region generates the highest transaction value?
2. What is the average transaction amount by account type?
3. Which account type has the highest number of transactions?
4. What are the monthly transaction trends?

## Key Insights
- Savings accounts generated higher average transaction values.
- Asia recorded the highest transaction volume.
- Transaction activity increased in the second quarter.

## Files Included

banking_data.csv – sample banking dataset  
banking_analysis.sql – SQL queries used for analysis
